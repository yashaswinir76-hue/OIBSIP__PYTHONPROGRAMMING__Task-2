# BMI Calculator (Python Tkinter)

## 📌 Project Description

This project is a **GUI-based BMI (Body Mass Index) Calculator** built using **Python and Tkinter**. The application allows users to enter their **weight (kg)** and **height (m)** and calculates their BMI along with the corresponding **health category** such as Underweight, Normal, Overweight, or Obese.

---

## 🎯 Features

* Simple and user-friendly graphical interface
* Calculates BMI instantly
* Displays BMI category
* Error handling for invalid inputs
* Lightweight desktop application

---

## 🧮 BMI Formula

BMI is calculated using the formula:

BMI = Weight / (Height²)

Where:

* **Weight** is in kilograms (kg)
* **Height** is in meters (m)

---

## 🏷 BMI Categories

| BMI Range      | Category    |
| -------------- | ----------- |
| Less than 18.5 | Underweight |
| 18.5 – 24.9    | Normal      |
| 25 – 29.9      | Overweight  |
| 30 and above   | Obese       |
